5) „License key” vs „Workspace‑based role gating” (verdict + arhitectură)

Decizie: Default = Workspace‑based role gating. License key doar pentru white‑label/self‑host/offline.

De ce workspace‑gating?

Control exact pe seats, roluri, expirare, audit, SSO, RLS, minimă fricțiune la onboarding; telemetrie completă (ARPU, usage, policy hits). 

Pachetele de industrie se atașează direct la organizations.entitlements.

Când license key?

White‑label self‑host, medii izolate, parteneri care cer activare offline.

Implementare: license_keys(key, org_id/null, product, seats, expiry, status, signature) + online verification (HMAC + nonce), „kill switch”, „device binding”.

Fluxuri (pseudo):

Workspace: login → org_id → entitlements (cache 5 min) → UI gates.

License key: UI „Activate Pack” → POST /api/license/activate → semnătură + bind la org → re‑hydrate entitlements.

Afișează la checkout prețul anual = 10× lunar (2 luni gratis). Blochează stacking cu alte cupoane (vezi 2.6).

Upgrade instant (Creator→Pro; Pro→Enterprise) cu proration Stripe.

Downgrade intră la următorul ciclu; exportă bundle înainte de schimbare (buton dedicat). 

2.5. Overage/Fair‑Use

Definiește unitatea de cost: „run GPT test” + „tokens”.

Pro: include pachet generos; depășirile se throttle‑uiesc, nu se taxează automat (evită surprize).

Enterprise: overage contractual, cu volum & rate limit clar (SLA).

2.6. Cupoane & stacking

1 cupon activ / achiziție; no‑stack cu Annual n‑2 sau Founder Annual; valabilitate 14–30 zile; max 20% în afara campaniilor speciale.

2.7. Refund & anulare

Money‑back 7 zile doar la prima plată Pro/Enterprise (nu la reînnoiri).

Anulare oricând; acces până la finalul perioadei plătite; export garantat.

2.8. Taxe & valute

Prețuri afișate EUR; colectează TVA UE (OSS) pentru B2C; reverse charge pentru B2B cu VAT valid.

2.9. Proprietate & date

Artefactele create aparțin clientului; păstrezi loguri tehnice (telemetrie) pentru îmbunătățirea sistemului, aggregate/anonymized; DPA disponibil la Enterprise.

Retenție: Free/Creator local only; Pro/Enterprise cloud 90 zile istoric garantat (extensibil Enterprise). 

2.10. Suport & SLA

Free/Creator: help‑center + comunitate.

Pro: e‑mail 24–48h.

Enterprise: 8×5, 24h SLA inițial, opțional 24×5 cu addendum.

2.11. Schimbare prețuri & grandfathering

Anunță cu 30 zile înainte; clienții existenți rămân la vechiul preț 12 luni (grandfather) sau primesc credit echivalent.

2.12. Anti‑abuz & conformitate

1 cont/user; interzice reselling fără acord; enforcează rate limits; raportare abuz la security@; GDPR‑ready by design.

// /api/stripe/webhook
switch (event.type) {
  case 'checkout.session.completed':
  case 'customer.subscription.created':
  case 'customer.subscription.updated':
  case 'customer.subscription.deleted': {
    const sub = event.data.object as Stripe.Subscription
    const orgId = mapCustomerToOrg(sub.customer)
    const plan = mapPricesToPlan(sub.items.data)   // creator|pro|enterprise
    const seats = calcSeats(sub.items.data)        // enterprise: base 5 + extras

    break
  }
}

function baseFlagsForPlan(plan: 'free'|'creator'|'pro'|'enterprise'): FlagSet {
  if (plan==='free') return {
    canUseAllModules: false,
    canExportMD: false,
    canExportPDF: false,
    canExportJSON: false,
    canUseGptTestReal: false,
    hasCloudHistory: false,
    hasEvaluatorAI: false,
    hasAPI: false,
    hasWhiteLabel: false,
    hasSeatsGT1: false,
    canExportBundleZip: false,
    allowedModules: ['M01','M10','M18']
  }
  if (plan==='creator') return { ...free(), canUseAllModules:true, canExportMD:true, allowedModules:'ALL' }
  if (plan==='pro') return { ...creator(), canExportPDF:true, canExportJSON:true, canUseGptTestReal:true, hasCloudHistory:true, hasEvaluatorAI:true }
  return /* enterprise */ { ...pro(), hasAPI:true, hasWhiteLabel:true, hasSeatsGT1:true, canExportBundleZip:true }
}

Playbook de Discounting (CONTROLAT) — PROMPTFORGE™

Obiectiv dur: crește ARPU și cash‑flow fără canibalizare; susține „Pro = planul de adevăr”, „Enterprise = decoy de prestigiu + upsell”, „Creator = scut anti‑downsell”. Încorporează evaluator AI, export bundle și telemetrie ca pârghii de valoare, nu de preț. 
 
 

1) Principii de preț (hard rules)

Annual‑only = „n‑2”: preț anual setat structural la 10× lunar (≈‑16,7% ≃ „2 luni gratis”), fără cupon. Ex. Pro 49€/lună → 490€/an; Creator 19€/lună → 190€/an; Enterprise 299€/lună → 2.990€/an. 

Founder batch (Pro anual) = ‑30% doar primul an, limită 100. Niciun stacking în afara discountului anual (Founder se aplică peste anualul Pro).

Enterprise seats volum: numai pe linia „seats”, nu pe baza Enterprise:

25+ seats: ‑10% / seat

50+ seats: ‑20% / seat
Baza de 5 seats la 299€ rămâne neschimbată.

Industry Pack (licență pe verticală): niciodată sub 1.490€ net (floor dur). Lista de preț: 1.990€/an/org (default). Fără cupoane publice. 
 

Valoarea ≠ reducere: comunică diferențele de capabilități (export .pdf/.json/.bundle, GPT Test real, cloud history, Evaluator AI, white‑label, API). Discountul doar „de trecere” între stări, nu de poziționare. 
 

2) Matrice de eligibilitate & non‑stacking

Annual‑only: eligibil pentru toate planurile; implicit în prețul anual (nu cupon).

Founder (‑30% Pro anual): doar conturi noi; un singur redemption / persoană / card‑fingerprint; valabil 12 luni; nu se combină cu alt cupon; se combină implicit cu anual (e scopul).

Enterprise volum seats: aplicat automat la add‑on seat item când total_seats ≥ 25 (‑10%) sau ≥ 50 (‑20%); retroactive „pe toți seat‑ii” cât timp pragul e menținut.

Industry Pack floor: respinge orice reducere care ar duce totalul sub 1.490€; fără coduri publice; doar excepții contractuale aprobate manual.
