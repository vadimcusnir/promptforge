# Plan detaliat pentru animații PromptForge

## Animații obligatorii

### 1. Clepsidră 2D cu cuvinte care curg și dispar
- **Format:** SVG animat
- **Rezoluție:** 512×512px
- **Descriere:** O clepsidră minimalistă formată din două triunghiuri conectate, cu cuvinte care cad din partea superioară în partea inferioară și dispar.
- **Animație:** 
  - Cuvintele apar prin efect de typing în partea superioară
  - Cad prin gâtul clepsidrei
  - Se estompează și dispar în partea inferioară
- **Durată:** 2 secunde (loop continuu)
- **Interactivitate:** Pulsație la hover

### 2. Text "You're Bleeding Hours" cu fading stroke
- **Format:** SVG animat
- **Rezoluție:** 1024×512px
- **Descriere:** Text bold geometric cu efect de stroke care pulsează și se estompează, sugerând scurgerea timpului.
- **Animație:** 
  - Stroke-ul textului pulsează între vizibil și invizibil
  - Efect de "sângerare" cu linii roșii care curg în jos din text
- **Durată:** 3 secunde (loop continuu)
- **Interactivitate:** Intensificare la hover

### 3. 7 cercuri concentrice (7D), LED-uri pulse
- **Format:** SVG animat
- **Rezoluție:** 512×512px
- **Descriere:** Șapte cercuri concentrice cu puncte LED care pulsează în secvență.
- **Animație:** 
  - Cercurile rămân statice
  - LED-urile pulsează în cascadă de la interior spre exterior
- **Durată:** 2 secunde (loop continuu)
- **Interactivitate:** Schimbare direcție pulsație la hover

### 4. Tabel Before/After cu flip 2D + bară progres
- **Format:** SVG animat
- **Rezoluție:** 1024×512px
- **Descriere:** Un tabel minimalist cu două coloane (Before/After) care se rotește pentru a arăta diferențele, însoțit de o bară de progres.
- **Animație:** 
  - Tabelul se rotește pe axa Y pentru a arăta diferențele
  - Bara de progres se umple sincronizat cu rotația
- **Durată:** 2 secunde (flip complet)
- **Interactivitate:** Declanșare flip la hover

### 5. Buton brutalist "Start the Forge" cu flash și linie roșie
- **Format:** SVG animat
- **Rezoluție:** 512×256px
- **Descriere:** Buton cu design brutalist cu text "Start the Forge", efect de flash la hover și o linie roșie care se desenează în jurul butonului.
- **Animație:** 
  - Flash subtil al butonului
  - Linie roșie care se desenează progresiv în jurul butonului
- **Durată:** 1.5 secunde (loop continuu)
- **Interactivitate:** Intensificare flash la hover

# Analiză stil vizual e2b.dev pentru animații PromptForge

## Caracteristici stil brutalist-flat + protocol cușnirian

### Elemente vizuale
- Interfață minimalistă cu elemente geometrice simple
- Borduri colorate în jurul elementelor (galben, roșu, mov)
- Text monospace pentru interfață și date
- Contrast puternic între fundal (negru) și elemente (culori vibrante)
- Elemente numerotate (1, 2, 3...) în colțurile componentelor
- Linii punctate pentru conexiuni între elemente
- Simboluri și glyphs minimaliste
- Ferestre/containere cu aspect de terminal sau consolă
- Grafice și vizualizări de date minimaliste (bare, puncte)

### Animații observate
- Pulsații subtile ale elementelor
- Tranziții simple (fade, slide)
- Efecte de typing/console pentru text
- Elemente care clipesc (blinking) subtil
- Linii care se desenează progresiv

### Paletă de culori
- Fundal: negru mat (#000000)
- Accente: auriu neon (#FFD700), roșu (#FF0000)
- Text: alb murdar (#E0E0E0), gri plumb (#5a5a5a)
- Borduri colorate pentru diferite secțiuni

### Tipografie
- Fonturi bold geometrice pentru titluri
- Monospace light pentru detalii și date
- Spațiere consistentă
- Aliniere precisă

# Idei pentru primul set suplimentar de animații

1. **Glitch Line Divider** – o linie orizontală care „tremură" și se rupe 1–2 frame-uri, apoi revine stabilă. Folosit ca separator de secțiuni.

2. **Cursor Prompt** – un cursor care tastează un singur cuvânt, apoi șterge, apoi tastează altul („Prompt → System → Forge").

3. **Hexa-Core Pulse** – un hexagon împărțit în 6 segmente, fiecare pulsează pe rând, ca un ciclu operațional.

4. **Circuit Board Flat** – o placă simplă de circuit cu trasee care se aprind pe rând (neon roșu → auriu).

5. **Clock Minimal** – un ceas 2D fără cifre, doar cu limbile; secundarul accelerează brusc, simbolizând „timp pierdut".

6. **Toggle Switch** – un comutator brutalist (pătrat) care trece din OFF (roșu) în ON (auriu).

7. **Loading Bar ASCII** – o bară de progres cu caractere [■■■■□□□□] care se umple în loop.

8. **Chip Data Flow** – un pătrat cu săgeți bidirecționale care pulsează, ca fluxul de date între module.

9. **Triangle Cascade** – trei triunghiuri aliniate, fiecare cade ca un domino și pulsează.

10. **Module Slot** – o cutie deschisă unde intră „cartele" (pătrate mici) care se fixează prin slide.

11. **LED Status** – un cerc mic care clipește: verde (OK) → roșu (Error) → galben (Processing).

12. **ASCII Terminal** – text care apare în stil >_ RUN MODULE… [OK]. Fade out, reloop.

13. **Fragment Bar** – bare verticale care se umplu în succesiune, ca un equalizer.

14. **Magnetize Effect** – două linii paralele atrag o formă mică (pătrat), apoi o eliberează.

15. **Network Nodes** – 5 puncte conectate prin linii, fiecare pulsează pe rând ca un pachet transmis.

16. **Binary Flip** – cifre 0 și 1 care se inversează rapid, până apare un cuvânt („Forge").

17. **Key & Lock** – o cheie minimalistă intră într-o fantă și se aprinde lock-ul (roșu → auriu).

18. **Shield Protocol** – un scut plat cu glitch care se repară singur (fractură → refacere).

19. **Circle Scan** – un cerc cu linie de scanare care trece 360° și lasă highlight neon.

20. **Error → Fix** – X roșu care se transformă prin morph într-un ✔ verde.

21. **Data Cube Rotation** - Un cub minimalist care se rotește lent, cu fețele colorate în negru și auriu, reprezentând transformarea datelor.

22. **Neural Network Pulse** - Noduri interconectate care pulsează secvențial, cu linii de conexiune care se aprind pe măsură ce "informația" călătorește.

23. **Command Line Typing** - Un prompt de comandă care tastează automat comenzi precum "forge --init", "prompt --generate", urmat de "success".

24. **Fractal Expansion** - O formă geometrică simplă care se replică și se extinde în modele fractale, apoi se contractă înapoi.

25. **Binary Rain** - Cifre binare (0 și 1) care cad vertical în stilul "Matrix", dar cu estetica brutalistă și culorile specificate.

## Specificații pentru toate animațiile:

- **Stil**: Brutalist-flat, vectorial, fără umbre
- **Paletă de culori**: negru mat (fundal), auriu neon (#FFD700), roșu (#FF0000), alb murdar (#E0E0E0), gri plumb (#5a5a5a)
- **Format**: SVG animat
- **Durată**: 1.5-3 secunde per loop
- **Efecte**: fade, pulse, glitch, flip, morph
- **Rezoluție**: 512×512px sau 1024×1024px

## Utilizare

Toate animațiile sunt create ca SVG-uri animate și pot fi integrate direct în site-ul web prin:

1. **Includere directă** - Copiați codul SVG direct în HTML:
   ```html
   <div class="animation-container">
     <!-- Copiați conținutul fișierului SVG aici -->
   </div>
   ```

2. **Referință externă** - Includeți SVG-ul ca imagine:
   ```html
   <img src="path/to/animation.svg" alt="Descriere animație">
   ```

3. **Încărcare cu JavaScript** - Pentru control programatic:
   ```javascript
   fetch('path/to/animation.svg')
     .then(response => response.text())
     .then(svgContent => {
       document.getElementById('animation-container').innerHTML = svgContent;
     });
   ```

4. **Încorporare în CSS** - Pentru elemente de fundal:
   ```css
   .element {
     background-image: url('path/to/animation.svg');
     background-repeat: no-repeat;
     background-position: center;
   }
   ```

## Accesibilitate

Toate animațiile respectă `prefers-reduced-motion` pentru utilizatorii care preferă mai puțină mișcare:

```css
@media (prefers-reduced-motion: reduce) {
  .animation-element {
    animation: none;
  }
}
```

## Personalizare

Animațiile pot fi personalizate prin modificarea variabilelor CSS din fiecare fișier SVG:

```css
:root {
  --black: #000000;
  --gold: #FFD700;
  --red: #FF0000;
  --white: #E0E0E0;
  --gray: #5a5a5a;
}
```
